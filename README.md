# Aion Sovereign

**Japan's Public Data, Structured for Developers**

365 API endpoints providing structured JSON access to Japanese government and public data.

## Overview

Aion Sovereign is a B2B data API for developers and businesses that need Japanese market data in English.

- 365 API endpoints across 20+ categories
- Sources: EDINET, e-Gov, e-Stat, JMA, EPO, and more
- Clean JSON responses in English
- 24/7 availability
